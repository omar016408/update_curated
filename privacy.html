Data Collection:

Our analytics are collected at the cookie and DNS level for security purposes in order to inhibit malicious actors from stealing bandwhich, our intellectual property, and to prevent web scraping that could be used to serve our consumers a forged version of our website. Our proxy provider for these needs is Cloudflare. Cloudflare uses a framework called strictly necessary cookies to limit your web footprint. This website contains only one cookie and it is the Cloudflare cookie. We are hosted through GitHub Pages so we can only serve static content with no server-side analytics whatsoever. GitHub "may collect User Personal Information from visitors to your GitHub Pages website, including logs of visitor IP addresses, to comply with legal obligations, and to maintain the security and integrity of the Website and the Service."
